STMTNIN：Spatial-Temporal Masked Transformer Networks with Informer

面向算法岗面试的论文复习笔记：聚焦数据格式、任务定义、模型架构、核心深度学习机制、实验设计与可回答的总结点。
Paper Core

一句话概括

STMTNIN 用 Masked Spatial Transformer 建模传感器间全局空间依赖，用 Informer 建模长程时间依赖，解决 1-2 小时交通速度多步预测中误差随预测步长累积的问题。

Main Motivation

要解决什么

交通速度同时存在空间相关性和时间周期性。传统 GCN/RNN 或 STTN 在短期预测有效，但在 60-120 分钟多步预测中容易出现长程依赖不足、远距离空间关系利用不充分、误差增长较快。

Key Idea

核心创新

空间 Mask 注意力温度 τ 蒸馏重加权Chebyshev GCN 局部图卷积Informer ProbSparse融合门控

1. 数据格式特征与任务目标

输入 / 输出定义

图结构道路网络表示为 G=(V,E,A)，N=|V| 个传感器节点，A∈RN×N 表示道路连接或距离邻接。输入张量X∈RN×L×M，N 为传感器数，L 为历史时间步，M 为每个节点的观测特征数；本文主要用速度，M=1。输出张量Y∈RN×H，预测未来 H 个时间步的交通速度。采样间隔5 分钟。默认窗口L=12，即使用过去 60 分钟预测未来多步。预测范围15/30/45/60/90/120 分钟，对应 H=3/6/9/12/18/24。
Y = F(X; G)

数据集与任务属性

数据集规模与特点用途
Seattle323 个传感器，覆盖 I-5、I-405、I-90、SR-520；5 分钟采样。验证城市路网速度预测。

面试记忆：这是典型的图上的多变量时间序列回归：节点是传感器，边/邻接矩阵来自道路连接或距离，目标是多步交通速度预测。

2. 总体架构：Input Embedding → Spatial-Temporal Layer × n → Forecast Layer

论文 Figure 1：STMTNIN 架构。底部 1×1 卷积升维；中间堆叠空间-时间层；顶部 1×1 卷积输出预测。

前向流程

1
输入嵌入
1×1 Conv 将 X 从 M 维投影到 C 维。
2
空间模块
空间位置嵌入 + Masked Self-Attention + Chebyshev GCN + 门控融合。
3
时间模块
时间位置嵌入 + Informer ProbSparse Attention + Distilling。
4
预测层
两层 1×1 Conv 映射到未来 H 步速度。

为什么是“空间-时间”分块？

  • 空间上：传感器间既有局部邻接，也可能存在远距离强相关；仅用 GCN 可能漏掉远距离关系。
  • 时间上：速度序列有日常出行规律、拥堵扩散与长程依赖；标准 Transformer 的 O(L²) 注意力在长序列上计算成本高。
  • 因此空间侧用 Masked Transformer + GCN，时间侧用 Informer。

3. 核心模块一：Input Embedding 与空间位置嵌入

1×1 Convolution 输入升维

原始 X∈RN×L×M 特征较低维，先通过 1×1 卷积映射到隐藏通道 C，得到适合注意力、GCN 与后续 FFN 处理的表示。

面试解释：这里的 1×1 Conv 等价于对每个节点-时间位置做共享线性投影，作用是通道升维/特征混合，不改变 N 和 L。

Trainable Spatial Embedding

论文使用可学习空间嵌入 DS∈RN×L×C。其基础来自可训练邻接矩阵 Ã∈RN×N，初始化为归一化物理邻接 A，并在训练中自适应调整。

E^S = Ã W_A ∈ R^{N×C} D^S = Repeat(E^S, L) ∈ R^{N×L×C} X^S = X + D^S

这样既保留物理路网先验，又允许模型学习数据驱动的空间修正。

4. 核心模块二：Spatial Masked Transformer

4.1 Masked Self-Attention：全局空间关系 + 弱相关过滤

空间自注意力在同一时间步上建模 N 个传感器之间的全局关系。普通 self-attention 会给所有节点分配权重，包含大量弱相关或噪声关系。STMTNIN 加入 mask 层，将低于阈值 ε 的注意力 score 置为 -∞，softmax 后对应权重为 0。

Q^S = X^S W_q^S, K^S = X^S W_k^S, V^S = X^S W_v^S S^S = softmax( mask(Q^S (K^S)^T) / sqrt(C) τ ) mask(z) = z, if z ≥ ε = -∞, if z < ε
  • 目的：抑制弱空间相关，突出对目标节点有贡献的传感器。
  • 优势：能捕获远距离但高度相关的节点，不局限于物理邻居。
  • 风险：mask 可能过早过滤有用信息，因此引入温度 τ 缓解训练不稳定。

4.2 温度 τ：注意力分布的平滑与锐化

τ 来自知识蒸馏和 Boltzmann 分布思想。较大的 τ 会让 softmax 分布更平滑，降低早期训练中注意力过度尖锐导致的不稳定；训练后期 τ 退火到 1，使注意力聚焦强相关传感器。

面试记忆：τ 大 → 分布平滑、探索更多节点、梯度更稳定；τ 小/退火到 1 → 分布更尖锐、强调强相关节点。

输出形式

U^S = ReLU( S^S V^S + X^S ) W_0^S W_1^S + S^S V^S

该式体现了 attention 输出、残差连接与前馈投影的组合，用于增强非线性表达并稳定训练。

4.3 Chebyshev GCN：局部邻域图卷积

Masked attention 强在全局依赖，Chebyshev GCN 强在局部图结构。Chebyshev 多项式用 K 阶近似图卷积，避免显式特征分解，能捕获 K-hop 局部邻域。

g_θ * X ≈ Σ_{k=0}^{K-1} θ_k T_k(L̃) X L = I_N - D^{-1/2} A D^{-1/2} L̃ = 2L / λ_max - I_N
  • Tk(·)：第 k 阶 Chebyshev 多项式。
  • θk：可学习滤波系数。
  • K：最大邻域阶数，决定局部感受野。

4.4 Fusion Layer：全局注意力与局部 GCN 的门控融合

模型不是简单相加，而是用 sigmoid 门控自适应决定每个位置更依赖 masked attention 还是 GCN。

g = σ( f_s(U^S) + f_g(X_g) + b ) Y^S = g ⊙ U^S + (1-g) ⊙ X_g

其中 US 是 masked self-attention 输出，Xg 是 Chebyshev GCN 输出。

面试回答：门控融合解决“远距离全局相关”和“局部道路邻接”两种空间信息源的权衡问题，类似 learnable mixture。

5. 核心模块三：Temporal Informer

5.1 时间位置嵌入

空间模块输出 YS 后，加入可学习时间位置嵌入 DT∈RN×L×C。DT 由时间索引 one-hot 初始化并线性投影到 C 维。

X^T = Y^S + D^T

作用是让模型区分序列中的位置，学习时间规律、周期性与趋势。

5.2 ProbSparse Self-Attention

Informer 的核心是 ProbSparse self-attention。它认为长序列中只有少数 query 对全局依赖贡献大，因此只选 top-u 个重要 query 计算注意力，降低标准 Transformer 的二次复杂度。

Q^T = X^T W_q^T, K^T = X^T W_k^T, V^T = X^T W_v^T M(q_i,K) = max_j(q_i k_j^T / sqrt(C)) - mean_j(q_i k_j^T / sqrt(C)) u = c · ln(L_Q) S^T = softmax( Q̄^T (K^T)^T / sqrt(C) )
  • M(qi,K) 衡量 query 的“稀疏重要性”。
  • top-u query 参与主要 attention 计算。
  • 适合长序列预测，减少无效注意力计算。

5.3 Self-attention Distilling

Informer 进一步用 Conv1d + ELU + MaxPool 做 distilling，逐步聚合显著时间模式并压缩冗余。

U^T = MaxPool( ELU( Conv1d(S^T) ) )

在本文中，Informer 有两个作用：捕获长程时间依赖；缓解 mask 层可能带来的异常值/不稳定问题。

5.4 预测层

堆叠 n 个 spatial-temporal layer 后，Forecast Layer 使用两层 1×1 Conv 将隐藏表示映射到未来 H 步预测。

U^T ∈ R^{N×L×C} → Forecast 1×1 Conv × 2 → Y ∈ R^{N×H}
注意：论文默认 n=2，C=64，空间与时间模块的注意力头数 h=2。

6. 与相关方法的区别

方法类型典型代表主要能力局限STMTNIN 的改进点
RNN/线性时间模型LSTM, DLinear, NLinear建模时间序列趋势或短中期依赖。基本忽略空间传感器关系。显式建模空间图关系和全局空间注意力。
GCN + RNN/CNNT-GCN, STGCN, DCRNN, Graph WaveNet利用图结构建模局部或扩散式空间依赖。远距离空间关系、长程时间关系可能不足。Masked Transformer 捕获全局空间相关，Informer 捕获长程时间依赖。
Transformer 时空模型STTN自注意力可建模全局依赖。长预测范围下误差仍增长；空间 attention 可能包含弱相关噪声。mask 过滤弱相关，τ 控制注意力锐度，Informer 稳定长预测。

7. 实验设置与总体结论

实验设置

  • 数据预处理：Z-score normalization。
  • 训练：MSE loss，Adam optimizer，初始学习率 0.001，weight decay 0.001。
  • batch size：32；epoch：20。
  • 默认超参：L=12，M=1，C=64，n=2，h=2，ε=0.001，τ 初始为 2，Informer sampling factor c=5。
  • 指标：MAE、RMSE、MAPE。

总体实验结论

  • STMTNIN 在 PeMSD7(M)、PeMSD7(L)、Seattle 上整体优于多数基线，尤其在 60-120 分钟中长期预测上优势更明显。
  • 时间-only 模型整体弱于时空模型，说明传感器空间依赖对交通预测必要。
  • 与 STTN 相比，STMTNIN 长预测范围误差增长更慢，说明 masked spatial attention + Informer 提升了稳定性。
  • 短期预测上，部分图模型如 DCRNN 或 Graph WaveNet 可能局部竞争，但长期误差累计更明显。

消融实验结论

变体含义观察
STTN基础时空 Transformer。可建模全局依赖，但长预测仍有误差增长。
STTN(mask)只加入空间 mask。能集中空间注意力，部分指标改善；但长期 RMSE 稳定性可能不足。
STTN(informer)只替换时间模块为 Informer。更稳定地降低长期误差，缓解异常值影响。
STMTNINmask + Informer。整体最优或具有竞争力，说明两者互补。

局限与未来工作

  • 严重拥堵或突发事件场景仍有提升空间。
  • 未来可加入天气、事故、节假日等外部因素，提高高峰期和异常状态预测能力。
  • 对于超大规模路网，空间 attention 仍需关注计算成本与 mask 策略鲁棒性。

8. 面试高频问答速记

Q1：STMTNIN 的核心贡献是什么？
它将空间侧的 Masked Transformer 和时间侧的 Informer 结合起来。空间 mask 过滤弱相关传感器并突出强依赖；Informer 用 ProbSparse attention 捕获长程时间依赖，提升 1-2 小时多步预测稳定性。
Q2：为什么同时用 Masked Self-Attention 和 Chebyshev GCN？
二者建模粒度互补。Masked Self-Attention 捕获全局、远距离但相关的传感器关系；Chebyshev GCN 捕获基于道路邻接的 K-hop 局部结构。融合门控让模型自适应选择两类空间信息。
Q3：mask 层怎么工作？
先计算空间 attention score，再将低于阈值 ε 的 score 置为 -∞，softmax 后其权重变为 0，从而过滤弱相关节点，减少噪声注意力。
Q4：τ 的作用是什么？
τ 是 softmax 温度。较大 τ 会平滑注意力分布，使训练早期更稳定；随后退火到 1，让分布更尖锐，聚焦强相关传感器。
Q5：Informer 为什么适合长时序预测？
Informer 的 ProbSparse attention 只选择最重要的 top-u queries 参与主要计算，降低标准 Transformer O(L²) 注意力成本，并通过 distilling 聚合显著时间模式，适合长程依赖建模。
Q6：STMTNIN 和 STTN 的主要区别？
STTN 使用空间 Transformer 和时间 Transformer；STMTNIN 在空间 Transformer 中加入 mask 与温度重加权，并用 Informer 替换/增强时间模块，因此更关注强空间相关并提升长预测稳定性。
Q7：模型的损失和评价指标是什么？
训练使用 MSE loss；评价使用 MAE、RMSE、MAPE。MAE 衡量平均绝对误差，RMSE 对大误差更敏感，MAPE 衡量相对百分比误差。
Q8：这篇论文的算法岗讲解主线怎么组织？
先说任务是图上的多变量时间序列预测；再说难点是空间远距离依赖和长程时间依赖；然后介绍空间模块“全局 mask attention + 局部 ChebGCN + 门控融合”，时间模块“Informer ProbSparse + distilling”；最后总结实验显示长预测范围误差增长更慢。

9. 30 秒总结

STMTNIN 是一个交通速度多步预测模型。输入是 N 个传感器在过去 L 个时间步的速度序列 X∈RN×L×M，输出未来 H 步速度 Y∈RN×H。模型先用 1×1 Conv 升维，然后堆叠空间-时间层：空间层同时用 masked self-attention 捕获全局传感器依赖、用 Chebyshev GCN 捕获局部道路邻接，并通过门控融合；时间层用 Informer 的 ProbSparse attention 和 distilling 捕获长程时间模式。实验在 Seattle、PeMSD7(M)、PeMSD7(L) 上验证，整体在 60-120 分钟预测中更稳，消融说明 mask 和 Informer 互补。

整理用途：算法岗面试复习。重点是理解时空预测问题、Transformer/GCN/Informer 的组合方式，以及每个模块解决的具体瓶颈。
