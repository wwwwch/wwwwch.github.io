56. 合并区间

排序 贪心 区间合并

题意核心

给定若干区间 [start, end]，合并所有重叠区间， 返回互不重叠且能覆盖原输入的区间数组。

记忆图解

① 按左端点排序
② 逐个扫描
③ 能合并就扩右端点
④ 不能合并就加入答案

关键判断

情况 条件 操作
重叠 ans[-1][1] >= c[0] 更新右端点：max(ans[-1][1], c[1])
不重叠 ans 为空 或 ans[-1][1] < c[0] 当前区间作为新区间加入

示例

输入：[[1,3],[2,6],[8,10],[15,18]]
排序后不变。
[1,3] 与 [2,6] 重叠 → 合并为 [1,6]
输出：[[1,6],[8,10],[15,18]]

代码

class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        ans = []  # 存储合并后的区间
        
        # 1. 排序：按区间左端点升序
        intervals.sort(key=lambda x: x[0])
        # 2. 遍历合并
        for c in intervals:
            # 如果ans不为空且当前区间与最后一个区间重叠
            if ans and ans[-1][1] >= c[0]:
                ans[-1][1] = max(ans[-1][1], c[1])  # 合并：更新最后一个区间的右端点
            else:
                ans.append(c)   # 不重叠：将当前区间作为新区间加入
        
        return ans

易错点

  • 必须先排序：否则只看最后一个区间无法保证正确。
  • >= 表示端点相接也算可合并，例如 [1,4] 和 [4,5]。
  • 合并时只更新右端点，左端点已由排序保证最小。

复杂度

时间复杂度 O(n log n)，主要来自排序
空间复杂度 O(n)，答案数组

口诀：先按左端点排好序，只和答案最后一个区间比较。
