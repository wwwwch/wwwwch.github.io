239. 滑动窗口最大值 —— 单调队列解法

核心记忆：用“递减索引队列”让队首始终代表当前窗口最大值。

1

方法对比

方法 核心思路 时间复杂度 空间复杂度
暴力法 每滑动一次，遍历窗口内所有元素取最大值 O(n·k) O(1)
单调队列法 维护双端队列存储索引，保持队列内元素值单调递减，队首即为窗口最大值 O(n) O(k)
2

单调队列法（最优解）

核心思想：使用双端队列 deque 存储索引，保证队列中索引对应的元素值单调递减（队首最大）。

遍历数组，对每个元素 x（索引 i）执行四步：

1
队尾维护：移除队尾所有 ≤ x 的索引；它们被 x 遮蔽，不可能再成为最大值。
2
加入当前索引：将 i 加入队尾。
3
队首过期：若队首索引 < i - k + 1（窗口左边界），弹出队首。
4
记录答案：当窗口完全形成（i ≥ k-1）时，队首元素值即为当前窗口最大值。
3

代码实现

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        ans = [0] * (len(nums) - k + 1)  # 窗口个数
        q = deque()  # 双端队列

        for i, x in enumerate(nums):
            # 1. 右边入
            while q and nums[q[-1]] <= x:
                q.pop()  # 维护 q 的单调性
            q.append(i)  # 注意保存的是下标，这样下面可以判断队首是否离开窗口

            # 2. 左边出
            left = i - k + 1  # 窗口左端点
            if q[0] < left:  # 队首离开窗口
                q.popleft()

            # 3. 在窗口左端点处记录答案
            if left >= 0:
                # 由于队首到队尾单调递减，所以窗口最大值就在队首
                ans[left] = nums[q[0]]

        return ans
4

复杂度分析

时间复杂度：O(n) —— 每个元素最多入队、出队一次。
空间复杂度：O(k) —— 队列最多存储 k+1 个索引。
5

关键细节

细节 说明
存索引而非值 便于判断元素是否滑出窗口（比较索引与 left）。
队尾移除条件 ≤ x 相等时保留新索引（旧索引先过期，不影响结果）。
队首过期条件 < left 窗口左边界 left 包含在内，索引等于 left 仍属于窗口。
答案下标 第 left 个窗口（从0开始）的最大值存入 ans[left]。
6

一句话总结

◎

单调队列维护窗口内递减索引序列，队首始终是当前窗口最大值；每个元素入队出队一次，时间复杂度 O(n)，空间 O(k)。

记忆口诀：右边入时“去小留大”，左边出时“过期弹出”，窗口成形后“队首即答案”。
