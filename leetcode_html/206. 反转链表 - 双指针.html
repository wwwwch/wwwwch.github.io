🔗 206. 反转链表｜紧凑复习笔记

核心：逐个改变 next 指向。首选 双指针迭代；递归用于理解回溯与状态传递。

O(N)原地反转指针题防断链 / 防成环
存q = pre.next
反pre.next = cur
移cur / pre 前进

★总图：链表反转在做什么

1→2→3→None
1←2←3←None
一句话：遍历每个节点，把它的 next 从“指向后面”改成“指向前面”。

1双指针迭代法｜首选

状态
cur：已反转头；pre：当前待处理节点。
口诀
暂存下一 → 反向链接 → 双指针前进
复杂度
时间 O(N)；空间 O(1)。
q = pre.nextpre.next = curcur = prepre = q
易错：必须先保存 q，否则 pre.next = cur 后会丢失后续链表。
代码：双指针迭代 
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 双指针法：pre 指向当前处理节点，cur 指向已反转部分的头节点（即 pre 的新 next）
        # 初始时：
        # - pre 指向原链表头节点（待反转的第一个节点）
        # - cur 指向 None（表示已反转部分为空）
        cur = None
        pre = head
        
        # 遍历原链表，直到 pre 为 None（即所有节点都已反转）
        while pre is not None:
            # 1. 保存 pre.next，防止断链后丢失后续节点
            q = pre.next
            
            # 2. 将 pre 所指节点的 next 指向 cur，完成局部反转
            pre.next = cur  
            
            # 3. 更新 cur：将 cur 前进到 pre 位置，代表已反转部分的新头部
            cur = pre
            
            # 4. 更新 pre：移动 pre 到下一个待处理节点（之前保存的 q）
            pre = q
        
        # 最终 cur 指向原链表最后一个节点，即新链表的头节点
        return cur

2递归回溯法｜自底向上

思想
先递归到尾节点，回溯时让后继节点指回当前节点。
关键
head.next.next = head + head.next = None
复杂度
时间 O(N)；递归栈空间 O(N)。
head→next变为head←next
易错：head.next = None 不能漏，否则可能形成环。
代码：递归回溯 
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        new_head = self.reverseList(head.next)
        head.next.next = head  # 将后继节点指向当前节点
        head.next = None       # 断开原向后指针
        return new_head

3显式状态递归｜递归版双指针

思想
把 pre / cur 作为递归参数，递归到末尾后统一改指针。
终止
cur == None 时，pre 就是新头。
复杂度
时间 O(N)；递归栈空间 O(N)。
recur(pre, cur)recur(cur, cur.next)cur.next = pre
代码：显式状态递归 
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        # 调用辅助方法，传入前驱节点（None）和当前节点
        return self.recur(None, head)
    
    def recur(self, pre: ListNode, cur: ListNode) -> ListNode:
        """
        递归反转链表的辅助函数
        pre: 当前节点的前驱节点（反转后的 next 指向）
        cur: 当前节点
        return: 新链表的头节点（原链表尾节点）
        """
        if not cur:
            return pre  # 终止条件：到达原链表末尾，pre 即为新头

        head = self.recur(cur, cur.next)    # 递归处理后续节点

        cur.next = pre  # 将当前节点的 next 指向前驱节点，完成反转

        return head     # 返回最终的新头节点（始终是最后一个返回的 pre）
