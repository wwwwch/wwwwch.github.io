📚20.有效的括号 —— 栈解法总结

🎯问题本质

判断一个仅包含 (){}[] 的字符串中的括号是否正确匹配且顺序正确。

🔍核心思路

使用栈：

  • 遇到左括号 ( { [ → 压入栈
  • 遇到右括号 ) } ] → 检查栈顶是否为对应的左括号
  • 若匹配，弹出栈顶；若不匹配或栈空 → 无效
  • 遍历结束后，栈应为空

📊两种实现对比

方法 实现方式 特点 代码示例
法一 自定义 Stack 类 封装性好，但代码稍长 st = Stack()
法二 列表模拟栈 + 映射字典 简洁高效，推荐 stack = [] bracket_map = {')':'(', ...}

💻关键代码片段

法二：列表 + 映射（推荐）
def isValid(self, s: str) -> bool:
    bracket_map = {')': '(', '}': '{', ']': '['}
    stack = []

    for ch in s:
        if ch in bracket_map:                # 右括号
            if not stack or stack.pop() != bracket_map[ch]:
                return False
        else:                                 # 左括号
            stack.append(ch)

    return not stack                          # 栈空则有效
法一：自定义栈类（仅核心逻辑）
# 使用自定义 Stack 类（push, pop, size）
st = Stack()
for ch in s:
    if ch in '({[':
        st.push(ch)
    else:
        if st.size() == 0: return False
        top = st.pop()
        if (ch == ')' and top != '(') or \
           (ch == '}' and top != '{') or \
           (ch == ']' and top != '['):
            return False
return st.size() == 0

⚠️关键细节

细节 说明
栈空遇到右括号 直接返回 False
映射表 统一处理右括号与左括号的对应关系
最终检查 遍历完后栈必须为空，否则有多余左括号
时间复杂度 O(n)，每个字符处理一次
空间复杂度 O(n)，最坏情况下栈内全是左括号

💡一句话总结

栈保存左括号，遇右括号时栈顶必须匹配，遍历完栈为空则有效。
