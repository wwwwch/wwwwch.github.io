169. 多数元素｜Boyer-Moore 摩尔投票

目标：找绝对众数 出现次数 > ⌊n/2⌋ O(n) 时间 O(1) 空间 题目保证存在

核心记忆：抵消

多数元素
票数 > 一半
−
其它元素
总票数 < 一半
=
最后存活
多数元素
一对不同元素互相抵消；因为多数元素数量更多，最终留下的必是多数元素。

题意压缩

  • 输入：数组 nums，长度 n
  • 输出：出现次数 > ⌊n/2⌋ 的元素
  • 绝对众数：比其它所有元素加起来还多
  • 保证：答案一定存在，无需二次验证
易错点：不是普通最高频；必须超过半数。

变量含义

变量作用记忆
major当前候选多数元素台上候选人
count候选元素净支持票支持 − 反对

三步流程

1
count == 0
无候选，当前元素上台
2
num == major
同阵营，支持票 +1
3
num != major
不同阵营，互相抵消 -1

为什么正确

  • 删除任意两个不同元素，多数元素仍然是多数元素。
  • 候选不是多数元素时，会被真正多数元素持续抵消，直到 count = 0。
  • 真正多数元素数量超过其它元素总和，因此一定能存活到最后。

复杂度与结论

  • 时间复杂度：O(n)，一次遍历
  • 空间复杂度：O(1)，仅两个变量
  • 最优解：无需排序、哈希表或额外计数结构
口诀：不同抵消，票空换人，多数必胜。

完整代码

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        major = 0     # 当前候选多数元素
        count = 0     # 候选元素的净支持票数

        for num in nums:
            if count == 0:    # 1. 当前无候选 → 新候选上台
                major = num
            if num == major:  # 2. 相同元素 → 支持票+1
                count += 1
            else:             # 3. 不同元素 → 反对票-1（抵消）
                count -= 1

        return major
