136. 只出现一次的数字｜异或

题目核心

给定非空整数数组 nums： 一个元素出现 1 次 其余元素出现 2 次

目标：找出只出现一次的元素。

记忆点：成对元素可以被异或“抵消”，最后剩下单独元素。

异或性质

  • a ⊕ a = 0：相同数字抵消
  • 0 ⊕ a = a：0 是异或初始值
  • a ⊕ b = b ⊕ a：交换律
  • (a ⊕ b) ⊕ c = a ⊕ (b ⊕ c)：结合律

消除过程示例

nums = [4, 1, 2, 1, 2]

4 ⊕ 1 ⊕ 2 ⊕ 1 ⊕ 2
4 ⊕ (1 ⊕ 1) ⊕ (2 ⊕ 2)
4 ⊕ 0 ⊕ 0 = 4

算法步骤

  • 初始化 x = 0
  • 遍历数组，每个元素执行 x ^= num
  • 重复元素互相抵消
  • 返回最终的 x
易错点：ans = 0 不是随便取值，因为 0 ⊕ a = a。

复杂度

时间：O(n)
空间：O(1)

结论

所有数异或一遍，出现两次的数全部变成 0， 剩下的就是只出现一次的数字。

代码

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        x = 0
        for num in nums:
            x ^= num
        return x
