📚 73. 矩阵置零

🎯 题意压缩

元素为 0 ⇒ 该元素所在 整行 + 整列 全部置 0；要求 原地修改。
矩阵原地算法标记不能污染原数据核心：先标记，后置零

🧠 一图记忆流程

1️⃣ 扫描矩阵：发现 0
↓
2️⃣ 记录该 行 / 列 需要清零
↓
3️⃣ 再次扫描：按标记置零
↓
4️⃣ V2 最后单独处理 第一行 / 第一列

📊 方法对比

V1 辅助数组O(m+n)

标记：row_zero + col_zero

优点：直观，不易错

缺点：额外空间

V2 常量空间O(1)

标记：复用第一行 + 第一列

优点：空间最优

易错：首行首列必须先保存状态

🔍 V1：辅助数组法

O(m+n)
  • 行标记：row_zero[i]
  • 列标记：col_zero[j]
  • 两遍：先标记，再置零

💻 完整代码

def setZeroes(self, matrix: List[List[int]]) -> None:
    m, n = len(matrix), len(matrix[0])
    row_zero = [False] * m
    col_zero = [False] * n

    # 第一遍：标记
    for i in range(m):
        for j in range(n):
            if matrix[i][j] == 0:
                row_zero[i] = True
                col_zero[j] = True

    # 第二遍：置零
    for i in range(m):
        if row_zero[i]:
            matrix[i] = [0] * n      # 整行清零
        else:
            for j in range(n):
                if col_zero[j]:
                    matrix[i][j] = 0

🔍 V2：常量空间法

O(1)
  • 第一行：记录哪些列清零
  • 第一列：记录哪些行清零
  • 易错：先存 first_row_zero / first_col_zero，最后处理首行首列

💻 完整代码

def setZeroes(self, matrix: List[List[int]]) -> None:
    m, n = len(matrix), len(matrix[0])
    first_row_zero = any(matrix[0][j] == 0 for j in range(n))
    first_col_zero = any(matrix[i][0] == 0 for i in range(m))

    # 第一遍：用第一行和第一列做标记（跳过第一行第一列）
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][j] == 0:
                matrix[i][0] = 0
                matrix[0][j] = 0

    # 第二遍：根据标记置零（跳过第一行第一列）
    for i in range(1, m):
        if matrix[i][0] == 0:
            matrix[i] = [0] * n
        else:
            for j in range(1, n):
                if matrix[0][j] == 0:
                    matrix[i][j] = 0

    # 最后处理第一行和第一列
    if first_row_zero:
        for j in range(n):
            matrix[0][j] = 0
    if first_col_zero:
        for i in range(m):
            matrix[i][0] = 0

⚠️ 易错点速查

点记忆
顺序先标记，后置零，不能边扫边清
V2 标记区matrix[i][0] 标行，matrix[0][j] 标列
首行/列先用变量保存，最后清零
遍历范围V2 主体从 1 开始，跳过首行首列

💡 一句话背诵

V1 用额外数组记行列；V2 借第一行/列当标记区，首行首列先保存、最后处理。
