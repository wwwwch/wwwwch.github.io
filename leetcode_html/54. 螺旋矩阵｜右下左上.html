54. 螺旋矩阵

顺时针螺旋遍历矩阵，返回所有元素
核心：四边界模拟

题意 Goal

  • 输入 m × n 矩阵 matrix
  • 按 顺时针螺旋顺序 返回所有元素

直观图 3×3 Example

1①
2②
3③
4⑧
5⑨
6④
7⑦
8⑥
9⑤
方向循环
左 → 右
上 → 下
右 → 左
下 → 上 输出：[1,2,3,6,9,8,7,4,5]

核心变量 Boundary

l左边界
r右边界
t上边界
b下边界
每打印完一条边，对应边界向内收缩 1；若边界交错，说明遍历结束。

算法流程 Loop

  • 空值处理：if not matrix: return []
  • 初始化：l, r, t, b 与结果列表 ans
  • 循环打印：四个方向依次遍历，并收缩边界
  • 终止条件：任意边界交错立即 break

四方向速记 Print → Shrink → Check

① 左 → 右
打印第 t 行
t += 1；查 t > b
② 上 → 下
打印第 r 列
r -= 1；查 l > r
③ 右 → 左
打印第 b 行
b -= 1；查 t > b
④ 下 → 上
打印第 l 列
l += 1；查 l > r

边界表 易记

打印方向打印范围收缩结束判断
左 → 右l 到 rt += 1t > b
上 → 下t 到 br -= 1l > r
右 → 左r 到 lb -= 1t > b
下 → 上b 到 tl += 1l > r

复杂度 Cost

时间：O(MN) 额外空间：O(1)

完整代码 Keep All

class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        if not matrix:
            return []

        l, r, t, b = 0, len(matrix[0])-1, 0, len(matrix)-1
        ans = []

        while True:
            for i in range(l, r+1):
                ans.append(matrix[t][i]) # left to right
            t += 1
            if t > b:
                break

            for i in range(t, b+1):
                ans.append(matrix[i][r]) # top to bottom
            r -= 1
            if l > r:
                break
            
            for i in range(r, l-1, -1):
                ans.append(matrix[b][i]) # right to left
            b -= 1
            if t > b:
                break
            
            for i in range(b, t-1, -1):
                ans.append(matrix[i][l]) # bottom to top
            l += 1
            if l > r:
                break

        return ans

易错点 Pitfalls

  • 每走完一条边，必须 立刻检查 是否越界。
  • 判断顺序不能省略，否则单行、单列矩阵可能重复打印。
  • 反向遍历范围：range(r, l-1, -1)、range(b, t-1, -1)。

一句话记忆 Mnemonic

右下左上，走一边缩一边；缩完先判断，交错就结束。
