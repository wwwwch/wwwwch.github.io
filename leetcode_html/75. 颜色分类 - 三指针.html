75. 颜色分类 三指针 / Dutch Flag

题意

原地排序数组 nums：
0 红 → 1 白 → 2 蓝
  • 输入：[2,0,2,1,1,0]
  • 输出：[0,0,1,1,2,2]

核心结论

看到 2：交换后 idx 不动，因为换来的元素未知。
  • p1：指向 0 区末尾之后
  • idx：遍历无序区
  • p2：指向 2 区开头之前

四区域划分

[0, p1) = 0 [p1, idx) = 1 [idx, p2] = 未知 (p2, n-1] = 2

三种情况

nums[idx] == 0 交换 p1 与 idx
p1++
idx++
nums[idx] == 1 已在 1 区
不交换
idx++
nums[idx] == 2 交换 idx 与 p2
p2--
idx 不动

指针移动表

当前值 操作 p1 idx p2
0 swap(p1, idx) +1 +1 不变
1 跳过 不变 +1 不变
2 swap(idx, p2) 不变 不变 -1
循环条件：while idx <= p2
因为 idx == p2 时仍有一个未知元素要处理。

代码完整保留

class Solution:
    def sortColors(self, nums: List[int]) -> None:
        p1, idx, p2 = 0, 0, len(nums)-1
        while idx<=p2:
            if nums[idx] == 0:
                nums[p1], nums[idx] = nums[idx], nums[p1]
                # 跟自己交换或者和1区的交换，都为已知的，idx直接+1
                p1 += 1
                idx += 1
            elif nums[idx] == 2:
                nums[p2], nums[idx] = nums[idx], nums[p2]
                # p2指向了无序区，未知，所以要不变idx，继续判断
                p2 -= 1
            else: # 指针本来就指向1区后一个位置，所以直接+1即可
                idx += 1
