347. 前 K 个高频元素

目标：返回数组中频率最高的 k 个元素，顺序不限。

哈希表 桶排序 时间 O(n) 空间 O(n) 倒序取高频

记忆句：数频率 → 按频率入桶 → 倒扫够 k 返回。

题意压缩

输入 nums = [1,1,1,2,2,3], k = 2
输出 [1,2]，也可 [2,1]
本质 找出现次数排名前 k 的元素。

步骤图

1统计频率
cnt = Counter(nums)
↓
2建立桶
buckets[c].append(x)，桶下标 = 频率。
↓
3倒序收集
从 max_cnt 到 1，len(ans) == k 立即返回。

关键结构

cnt元素 → 出现次数。
max_cnt最大频率，决定桶数组长度。
buckets[c]所有出现 c 次的元素。
ans答案数组，收集前 k 个高频元素。

速记卡

数Counter(nums)
桶max_cnt + 1
放buckets[c].append(x)
取reversed(buckets)

易错点

  • 桶下标就是频率，不是元素值。
  • buckets 长度为 max_cnt + 1，保留频率下标。
  • 必须倒序扫桶，高频优先。
  • 同桶频率相同，不需要排序。
  • 题目保证答案唯一，可在 len(ans) == k 时返回。

完整代码

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        # 第一步：统计每个元素的出现次数
        cnt = Counter(nums)
        max_cnt = max(cnt.values())

        # 第二步：把出现次数相同的元素，放到同一个桶中
        buckets = [[] for _ in range(max_cnt + 1)]  # 也可以用 defaultdict(list)
        for x, c in cnt.items():
            buckets[c].append(x)

        # 第三步：倒序遍历 buckets，把出现次数前 k 大的元素加入答案
        ans = []
        for bucket in reversed(buckets):
            ans += bucket
            # 注意题目保证答案唯一，一定会出现恰好等于 k 的情况
            if len(ans) == k:
                return ans

复杂度 & 结论

时间O(n)：统计 + 入桶 + 扫桶。
空间O(n)：哈希表 + 桶数组。
结论：Counter 统计频率，桶下标表示频率，再倒序收集前 k 个。
