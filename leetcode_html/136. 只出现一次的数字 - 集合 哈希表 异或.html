136. 只出现一次的数字

集合 / 哈希表 / 数学 / 异或：核心解法速记
✅ 最优：异或 O(n) O(1) ⚠️ 前提：其余数成对

📊 方法对比

方法 时间 空间 核心 推荐
集合添加/删除 O(n) O(n) 唯一性；成对删除 ⭐⭐⭐
哈希表计数 O(n) O(n) 统计频率，找 1 次 ⭐⭐⭐
数学法 O(n) O(n) 2×集合和 - 数组和 ⭐⭐
异或法 O(n) O(1) a⊕a=0；a⊕0=a ⭐⭐⭐⭐⭐

🔍 四种解法

1️⃣ 集合存储法

思路：首次出现加入；再次出现删除；最后剩下目标数。

def singleNumber(nums):
    seen = set()
    for num in nums:
        if num in seen:
            seen.remove(num)    # 成对出现，从集合中删除
        else:
            seen.add(num)       # 首次出现，加入集合
    return seen.pop()           # 集合中剩余的唯一元素
💡 注意：要求其他数字都出现两次；最终集合中只剩一个元素。
2️⃣ 哈希表计数法

思路：记录每个数字出现次数，再返回频率为 1 的数字。

def singleNumber(nums):
    count = {}
    for num in nums:
        count[num] = count.get(num, 0) + 1   # 统计频率
    for num, freq in count.items():
        if freq == 1:                        # 找到只出现一次的数字
            return num
💡 注意：两次遍历；空间与不同数字个数相关；通用性最好。
3️⃣ 数学法

思路：每个重复数在集合和中只算一次。

2 × sum(set(nums)) - sum(nums) = 只出现一次的数
def singleNumber(nums):
    return 2 * sum(set(nums)) - sum(nums)
💡 注意：严格依赖“其他数字都恰好出现两次”；Python 无整数溢出问题。
4️⃣ 位运算：异或法 最优解

思路：成对数字异或后抵消为 0，剩下唯一数字。

a ⊕ 0 = a a ⊕ a = 0 可交换 / 可结合 
def singleNumber(nums):
    result = 0
    for num in nums:
        result ^= num   # 依次异或所有数字
    return result
假设数组: [a, b, c, a, b] 计算过程: 0 ⊕ a = a a ⊕ b = a⊕b a⊕b ⊕ c = a⊕b⊕c a⊕b⊕c ⊕ a = (a⊕a)⊕b⊕c = 0⊕b⊕c = b⊕c b⊕c ⊕ b = (b⊕b)⊕c = 0⊕c = c ✅

⚠️ 适用条件 / 易错点

方法 适用 限制
集合 其他数字出现次数为偶数 额外 O(n) 空间
哈希表 任意频率统计 额外 O(n) 空间
数学 其他数字都出现两次 频率条件严格
异或 其他数字出现偶数次 不适合任意频率统计
记忆点：成对抵消是本题核心；面试优先写异或，解释三条性质。

📝 解题选择

1
面试 / 竞赛：首选 异或法，时间 O(n)、空间 O(1)。
2
通用统计：选 哈希表，扩展性强。
3
快速理解：集合法直观，数学法最短但限制严格。

💎 一句话总结

异或法用 a⊕a=0 抵消成对数字，最终留下唯一值；本题最优为 O(n) 时间、O(1) 空间。
