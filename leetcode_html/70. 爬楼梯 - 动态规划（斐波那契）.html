🪜 爬楼梯 Climbing Stairs

核心：每次爬 1 或 2 级，求到第 n 级共有多少种走法。

动态规划 斐波那契 递归会超时 LeetCode 70

🧠 记忆核心

f(n) = f(n-1) + f(n-2)
到 n-1 级
再走 1 步
+
到 n-2 级
再走 2 步
=
到 n 级
总方法数
边界f(1)=1，f(2)=2；DP 也可设 dp[0]=1, dp[1]=1。
本质最后一步只有两种来源：从 n-1 来，或从 n-2 来。
结论最优写法：滚动数组，时间 O(n)，空间 O(1)。

⚠️ 易错点

  • 暴力递归重复计算严重：如 f(5) 会多次算 f(3)、f(2)。
  • dp 长度通常设为 n+1，方便访问 dp[n]。
  • dp[0]=1 表示「不走也是一种起点状态」，不是爬 0 级的题目答案混淆。
  • 滚动数组中 a,b = b,a+b 是同时赋值，不会先覆盖 a。
记忆口诀：看「最后一步」：走 1 步来自 n-1，走 2 步来自 n-2。

📊 四种解法对比

方法 思路 时间 空间 评价
暴力递归 直接套 f(n)=f(n-1)+f(n-2) O(2^n) O(n) 会超时
记忆化递归 递归 + 缓存子问题 O(n) O(n) 避免重复
DP 数组 自底向上填表 O(n) O(n) 清晰
滚动数组 只保留前两个状态 O(n) O(1) 最优

1）暴力递归 O(2^n)
class Solution: 
    def climbStairs(self, n: int) -> int:
        # 基础情况：只有1级台阶时，只有一种走法
        if n == 1:
            return 1
        # 基础情况：有2级台阶时，有两种走法（1+1 或 2）
        if n == 2:
            return 2
        
        # 递归关系：到达第n级台阶的方式数 = 
        # 到达第(n-1)级的方式数（再走1步） + 到达第(n-2)级的方式数（再走2步）
        return self.climbStairs(n - 1) + self.climbStairs(n - 2)
2）记忆化递归 O(n) / O(n)
class Solution:
    def climbStairs(self, n: int) -> int:
        # 创建记忆数组 N，长度为 n+1，以便使用下标 1 到 n 存储对应台阶的方案数
        # 说明：数组下标从 0 开始，我们让 N[0] 空出不使用（或视为第 0 级台阶），
        # 第 1 级台阶对应 N[1]，第 n 级台阶对应 N[n]，因此需要 n+1 的长度
        N = [0] * (n + 1)
        return self.climbStairs_N(n, N)
    
    def climbStairs_N(self, n: int, N: List[int]) -> int:
        """
        记忆化递归：计算到达第 n 级台阶的方法数
        若结果已缓存，则直接返回；否则递归计算并保存
        """
        # 如果已计算过该子问题，直接返回缓存结果
        if N[n] > 0:
            return N[n]
        
        # 基础情况
        if n == 1:
            N[n] = 1
        elif n == 2:
            N[n] = 2
        else:
            # 递归关系：f(n) = f(n-1) + f(n-2)
            N[n] = self.climbStairs_N(n - 1, N) + self.climbStairs_N(n - 2, N)
        
        return N[n]
3）DP 数组 O(n) / O(n)
class Solution:
    def climbStairs(self, n: int) -> int:
        # 创建 DP 数组，dp[i] 表示爬到第 i 级台阶的方法数
        # 长度为 n+1 是为了方便索引：0 到 n 均可访问
        dp = [0] * (n + 1)
        
        # 初始状态：
        # dp[0] = 1：表示在地面（第 0 级）有一种方式（不走也算起点）
        # dp[1] = 1：爬到第 1 级只能走一步，因此只有一种方法
        dp[0] = 1
        dp[1] = 1
        
        # 从第 2 级台阶开始，递推计算每一级的方案数
        for i in range(2, n + 1):
            # 当前台阶的方案数 = 
            #   从上一级走 1 步过来的方法数 (dp[i-1]) +
            #   从上两级走 2 步过来的方法数 (dp[i-2])
            dp[i] = dp[i - 1] + dp[i - 2]
        
        # 返回爬到第 n 级台阶的总方法数
        return dp[n]
4）滚动数组 O(n) / O(1)
class Solution:
    def climbStairs(self, n: int) -> int:
        # 滚动数组优化：只保留前两个状态，节省空间
        a, b = 1, 1
        
        # 从第 2 级台阶开始递推到第 n 级
        for _ in range(2, n + 1):
            # 新的 b = 原 b + 原 a → 即 dp[i] = dp[i-1] + dp[i-2]
            # 同时更新 a 和 b：向前滚动一步
            a, b = b, a + b
            # 解释：
            #   a 更新为原来的 b（即前一级台阶的方案数）
            #   b 更新为 a + b（即当前级的新方案数）
        
        return b
