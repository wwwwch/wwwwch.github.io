189. 轮转数组｜“负负得正”三次反转

目标：将 nums 向右轮转 k 位，原地完成。
原地 O(1) 时间 O(n) 三次反转

核心拆分

nums = A + B → 目标 = B + A
1234 567
A = 前 n-k 个；B = 后 k 个
↓
567 1234
记忆：右轮转 k 位 = 把最后 k 个元素搬到最前面。

三次反转图解

1234 567
① 整体反转
765 4321
② 反转前 k 个｜③ 反转剩余
567 1234

算法步骤

  1. k %= n：把 k 转成有效轮转次数。
  2. reverse(0, n-1)：A+B → rev(B)+rev(A)。
  3. reverse(0, k-1)：rev(B) → B。
  4. reverse(k, n-1)：rev(A) → A。
rev(rev(X)) = X
易错：第二次反转范围是 0 ~ k-1，第三次是 k ~ n-1。

记忆卡片

拆
A + B
整反
rev(B)+rev(A)
分反
B + A
场景
数组右轮转 k 位
关键
后 k 个元素作为 B
本质
交换两个连续子数组 A、B
复杂度
时间 O(n)；空间 O(1)

完整代码

class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        def reverse(i: int, j: int) -> None:
            while i < j:
                nums[i], nums[j] = nums[j], nums[i]
                i += 1
                j -= 1
        
        n = len(nums)
        k %= n

        reverse(0, n-1) # 把 nums=A+B 反转
        reverse(0, k-1) # 单独反转 rev(B)
        reverse(k, n-1) # 单独反转 rev(A), 数组变成 B+A
