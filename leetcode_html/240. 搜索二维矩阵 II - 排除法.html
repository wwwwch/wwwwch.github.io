240. 搜索二维矩阵 II 排除法O(m+n)

题目条件

  • 每行：从左到右升序
  • 每列：从上到下升序
  • 目标：判断 target 是否存在
核心：从 右上角 或 左下角 出发，每次排除一整行或一整列。

为什么选右上角？

  • 右上角：所在行的较大值，所在列的较小值。
  • 若当前值 < target：当前行都太小，向下。
  • 若当前值 > target：当前列都太大，向左。
易错：不能从左上或右下开始，因为只能判断整体最小/最大，无法稳定排除方向。

决策流程

→
返回 True
/

复杂度

时间：O(m+n)
空间：O(1)
  • 每次循环排除 一行 或 一列。
  • 最多排除 m+n−1 次。
  • 比暴力 O(mn) 更高效。

完整代码

class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        i, j = 0, n-1  # 从右上角开始

        while i<m and j>=0:  # 还有剩余元素
            if matrix[i][j] == target:
                return True  # 找到 target
            if matrix[i][j] < target:
                i += 1  # 这一行剩余元素全部小于 target，排除
            else:
                j -= 1  # 这一列剩余元素全部大于 target，排除

        return False

记忆口诀

右上开始：小了下移，大了左移；每步砍掉一整行/列。
