📚 560. 和为 K 的子数组

前缀和 + 哈希表｜连续子数组计数｜可含负数
核心：prefix[j]-prefix[i]=k 最优：O(n) 易错：先查后存
🎯 问题本质
统计数组中和为 k 的连续子数组个数。
关键词：连续、计数、可有负数 → 不能用普通滑动窗口。
🧠 记忆图
1遍历 num
累加当前前缀和
➜
2查 cur_sum-k
已有多少次
➜
3答案加次数
count += ...
➜
4记录 cur_sum
给后面用
prefix[j] - prefix[i] = k
⇒ prefix[i] = prefix[j] - k
📊 方法对比
暴力枚举
O(n²) / O(1)
固定起点 i，累加终点 j；直观但慢。
前缀和 + 哈希表
O(n) / O(n)
查 cur_sum-k 出现次数；一次遍历统计全部答案。
⚙️ 最优解步骤
  1. cur_sum 记录当前前缀和。
  2. prefix_count 存「前缀和 → 出现次数」。
  3. 每个元素：先查 prefix_count[cur_sum-k]，再存 cur_sum。
  4. 初始化 prefix_count[0]=1，覆盖从数组开头开始的子数组。
⚠️ 易错点速记
点记法
存次数相同前缀和可能来自多个起点。
0:1处理 cur_sum == k。
先查后存避免当前前缀和自己匹配自己。
负数可用前缀和差值法不依赖单调性。
💻 代码 1：暴力枚举
完整保留原代码块：
def subarraySum(self, nums, k):
    total = 0
    n = len(nums)
    for i in range(n):
        s = 0
        for j in range(i, n):
            s += nums[j]
            if s == k:
                total += 1
    return total
💻 代码 2：前缀和 + 哈希表
完整保留原代码块：
def subarraySum(self, nums, k):
    count = 0
    cur_sum = 0
    prefix_count = defaultdict(int)
    prefix_count[0] = 1

    for num in nums:
        cur_sum += num
        count += prefix_count[cur_sum - k]   # 查找能凑成k的前缀和
        prefix_count[cur_sum] += 1          # 记录当前前缀和

    return count
注意：若单独运行，需确保环境中已有 defaultdict；例如通常来自 collections。
💎 一句话结论
🔁子数组和 → 两个前缀和之差。
🧺哈希表记录过去的前缀和次数。
✅当前只需找 cur_sum-k，答案一次遍历得到。
