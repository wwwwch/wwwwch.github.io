55. 跳跃游戏｜贪心速记

1题意压缩

从 0 出发，nums[i] 是当前位置最大跳跃长度；判断能否到达 最后下标。

2核心图解

3贪心公式

A
状态：max_right = 当前能到达的最远下标。
B
可达才更新：若 i ≤ max_right，更新 max_right = max(max_right, i + nums[i])。
C
两种提前结束：i > max_right → False；max_right ≥ n-1 → True。

4完整代码

def canJump(self, nums: List[int]) -> bool:
    n = len(nums)
    max_right = 0                     # 当前能到达的最远下标

    for i in range(n):
        if i > max_right:             # 当前位置不可达
            return False
        max_right = max(max_right, i + nums[i])
        if max_right >= n - 1:        # 已能到达终点
            return True

    return True

5易错点

先判断 i 是否可达 可达才扩展 覆盖终点即成功 不需要 DP
关键顺序：先看 i > max_right，再更新 max_right。不可达位置不能拿来扩展。

6复杂度 & 方法对比

维度结论
时间O(n)：一次遍历
空间O(1)：只用变量
方法特点
贪心最优，维护最远距离
DPO(n²)，能做但冗余
反向贪心从后找最左可达点
