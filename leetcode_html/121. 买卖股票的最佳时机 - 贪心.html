📈 121. 买卖股票的最佳时机

一次遍历 贪心 O(n) / O(1)

🎯 题意压缩

数组 prices 表示每天股价，只能买卖一次，求最大利润；亏钱则返回 0。

🔑 核心记忆

边走边看最低价：当前价当卖出价，用 price - min_price 更新答案。

买入min_price
卖出price
利润max_profit

🧭 流程图式步骤

1

初始化：min_price = prices[0]，max_profit = 0先假设第 1 天买入

2

遍历每个 price每一天都尝试作为卖出日

3

price >= min_price：计算利润并更新最大值profit = price - min_price

4

price < min_price：更新最低买入价更便宜，未来更可能赚

5

返回 max_profit全程最大可得利润

💻 代码完整保留

Python
def maxProfit(self, prices: List[int]) -> int:
    min_price = prices[0]      # 当前最低买入价
    max_profit = 0             # 最大利润

    for price in prices:
        if price >= min_price:
            # 尝试卖出，更新最大利润
            profit = price - min_price
            max_profit = max(max_profit, profit)
        else:
            # 遇到更低价格，更新最低买入价
            min_price = price

    return max_profit

⚠️ 易错点

买入必须在卖出前：所以只维护当前之前/当前为止的最低价。
价格一直下降：max_profit 保持 0，不能返回负数。
min_price 和 max_profit 都是在遍历中动态更新。

🧠 变量速查

变量含义
min_price到当前为止的历史最低价
max_profit到当前为止可获得的最大利润
profit当前价格卖出的临时利润

✅ 复杂度 & 结论

时间O(n)：一次遍历
空间O(1)：常数变量
一句话：最低买入价 + 当前卖出价 = 更新最大利润
