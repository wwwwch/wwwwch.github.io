48. 旋转图像：两次翻转 = 顺时针旋转 90°

题意 & 约束

n × n 方阵 顺时针 90° 不能新建矩阵 O(1) 空间
(i, j) → (j, n - 1 - i)
  • 第 i 行 → 第 n - 1 - i 列
  • 第 j 列 → 第 j 行
  • 目标坐标：行列互换 + 列反向

核心结论

(i,j) → 转置 (j,i) → 行翻转 (j,n-1-i)
  • 转置：沿主对角线交换 matrix[i][j] 与 matrix[j][i]
  • 行翻转：每一行左右翻转，即关于垂直中轴翻转
  • 组合效果：顺时针旋转 90°
记忆：顺时针 = 先转置，再反转每一行

实现细节

  • 转置范围：只遍历主对角线下方，避免重复交换。
  • for j in range(i)：表示 j < i。
  • 行翻转：直接使用 row.reverse()，原地修改。
  • 时间复杂度：O(n²)
  • 空间复杂度：O(1)
易错点：不是“上下翻转”，而是每一行左右翻转。

记忆卡片

  • 坐标公式 (i,j) → (j,n-1-i)
  • 操作顺序 转置 → 行翻转
  • 原地修改 不返回新矩阵
  • 反例记忆 先列翻转再转置也可旋转，但不是本代码顺序
一句话：主对角线换位，行内左右倒序。

代码完整保留

class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        # 第一步：转置
        for i in range(n):
            for j in range(i):  # 遍历对角线下方元素
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

        # 第二步：行翻转
        for row in matrix:
            row.reverse()
