238. 除了自身以外数组的乘积｜前后缀分解

题意

给定整数数组 nums，返回 answer：

answer[i] = nums 中除 nums[i] 外所有元素乘积

示例：

nums = [1,2,3,4] → [24,12,8,6]

核心思想

前缀乘积后缀乘积不包含自身

左边乘积
pre[i]
当前位置
nums[i]
右边乘积
suf[i]
answer[i] = pre[i] × suf[i]
记忆：当前位置不参与计算，只要 左边所有数 × 右边所有数。

定义与递推

  • pre[i]：nums[0] 到 nums[i−1] 的乘积
  • suf[i]：nums[i+1] 到 nums[n−1] 的乘积
pre[i] = pre[i−1] × nums[i−1]
suf[i] = suf[i+1] × nums[i+1]
pre[0] = 1，suf[n−1] = 1

空子数组乘积规定为 1，便于递推。

示例拆解

复杂度

时间 O(n)空间 O(n)

完整代码

class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)

        # 前缀乘积：pre[i] 表示从 nums[0] 到 nums[i−1] 的乘积
        pre = [1] * n
        for i in range(1, n):
            pre[i] = pre[i-1] * nums[i-1]

        # 后缀乘积：suf[i] 表示从 nums[i+1] 到 nums[n−1] 的乘积
        suf = [1] * n
        for i in range(n-2, -1, -1):
            suf[i] = suf[i+1] * nums[i+1]
        
        return [p*s for p, s in zip(pre, suf)]

速记模板

  • 先从左到右：记录每个位置 左边乘积。
  • 再从右到左：记录每个位置 右边乘积。
  • 最后：对应相乘。
易错点：
  • pre[i] 不包含 nums[i]
  • suf[i] 不包含 nums[i]
  • 边界必须初始化为 1
