121. 买卖股票的最佳时机 —— 一次遍历解法

🎯 问题本质

给定一个数组 prices，只能买卖一次，求最大利润。

🔍 核心思想

一次遍历，记录历史最低价。
遍历过程中，假设在历史最低点买入，当前价格卖出，计算利润并更新最大值。

维护买入价min_price
当前卖出利润price - min_price
答案更新max_profit

📊 算法流程

步骤 操作 变量变化
1 初始化 min_price 为第一个元素，max_profit = 0 记录最低买入价
2 遍历每个 price 逐个判断
3 若 price ≥ min_price 计算利润 profit = price - min_price，更新 max_profit
4 否则（price < min_price） 更新 min_price = price
5 返回 max_profit 最终结果

💻 关键代码

Python 
def maxProfit(self, prices: List[int]) -> int:
    min_price = prices[0]      # 当前最低买入价
    max_profit = 0             # 最大利润

    for price in prices:
        if price >= min_price:
            # 尝试卖出，更新最大利润
            profit = price - min_price
            max_profit = max(max_profit, profit)
        else:
            # 遇到更低价格，更新最低买入价
            min_price = price

    return max_profit

⚙️ 复杂度分析

维度 性能
时间复杂度 O(n)（一次遍历）
空间复杂度 O(1)（常数变量）

🧠 变量含义

变量 含义
min_price 到当前为止的历史最低价
max_profit 到当前为止可获得的最大利润
profit 当前价格卖出的临时利润

💎 一句话总结

遍历数组，维护历史最低价，计算当前卖出利润，取最大值即得最佳时机。
