53. 最大子数组和｜前缀和 + 贪心 / 动态规划

题目核心

给定整数数组 nums，找出一个连续子数组，使其和最大，返回最大和。

连续子数组 至少 1 个元素 O(n) 不能为空
输入 [-2,1,-3,4,-1,2,1,-5,4]
输出 6
原因 [4,-1,2,1] 的和最大，为 6

一眼记忆

DP：以 i 结尾的最大和 = max(自己, 自己 + 前一个结尾最大和)

前缀和：当前最大子数组和 = 当前前缀和 - 历史最小前缀和
  • 动态规划：决定“要不要接上前面的子数组”。
  • 前缀和 + 贪心：用当前前缀和减去之前最小前缀和。
  • 两种方法时间复杂度都是 O(n)。

V1｜前缀和 + 贪心

核心思路

① 子数组和 = 两个前缀和之差
② 遍历时维护 当前前缀和 con_sum
③ 维护历史最小前缀和 min_pre
④ 用 con_sum - min_pre 更新答案
⚠️ 易错点：子数组不能为空，所以要先计算 con_sum - min_pre，再更新 min_pre。 类比股票：不能同一天买入又卖出。 
class Solution: 
    def maxSubArray(self, nums: List[int]) -> int:
        ans = con_sum = nums[0]
        min_pre = min(0, nums[0])

        for i in range(1, len(nums)):
            con_sum += nums[i]  # 当前的前缀和
            ans = max(ans, (con_sum - min_pre)) # 减去前缀和的最小值
            min_pre = min(min_pre, con_sum)     # 维护前缀和的最小值

        return ans

V2｜动态规划

状态定义

dp[i]：以 nums[i] 结尾的最大连续子数组和

状态转移

dp[i] = max(nums[i], dp[i-1] + nums[i])
  • nums[i]：从当前元素重新开始。
  • dp[i-1] + nums[i]：接上前面的最大子数组。
  • 最终答案：max(dp)。
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        dp = [0] * n
        dp[0] = nums[0]

        for i in range(1, n):
            dp[i] = max(nums[i], (dp[i-1] + nums[i]))
        
        return max(dp)

对比速记

方法 核心变量 记忆点
前缀和 + 贪心 con_sum, min_pre 当前前缀和 - 历史最小前缀和
动态规划 dp[i] 接前面 or 自己重新开始
✅ 结论：本题本质是“连续区间最大和”。 看到连续子数组最大/最小和，优先想到 DP 或 前缀和差值。
