15. 三数之和｜双指针 + 去重

题目目标

给定整数数组 nums，返回所有满足： a + b + c = 0 的不重复三元组。

排序 双指针 去重 O(n²)
示例：
输入：[-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]

核心思路

1 排序
2 枚举 k
3 双指针 i / j
4 去重
  • 先排序，方便移动指针和跳过重复值。
  • 固定 nums[k]，转化为： nums[i] + nums[j] = -nums[k]
  • total < 0：左指针右移； total > 0：右指针左移。

去重规则

位置 做法
第 1 个数 k > 0 and nums[k] == nums[k-1] → 跳过
第 2 个数 找到答案后，跳过连续相同的 nums[i]
第 3 个数 找到答案后，跳过连续相同的 nums[j]
易错点：去重必须在找到一个合法三元组后处理内层重复值。

剪枝优化

  • 最小和 > 0：
    nums[k] + nums[k+1] + nums[k+2] > 0
    后面只会更大，直接 break。
  • 最大和 < 0：
    nums[k] + nums[-2] + nums[-1] < 0
    当前 k 太小，直接 continue。
  • nums[k] > 0：排序后后面都 ≥ 0，不可能凑 0。

复杂度

  • 时间复杂度：O(n²)
  • 空间复杂度：O(1)
  • 返回结果不计入；忽略排序栈开销。

记忆口诀

  • 先排序，定一位。
  • 双指针，夹逼走。
  • 小了左移，大了右移。
  • 命中收集，左右去重。
  • 最小大于零 break，最大小于零 continue。

完整代码

class Solution: 
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        if n < 3:
            return []  # 元素不足3个，不可能有解
        
        nums.sort()  # 关键步骤：排序后可以方便去重和使用双指针
        result = []
        
        for k in range(n - 2):  # k遍历到倒数第三个元素
            # 优化1: 如果第一个数已经大于0，后面不可能有解
            if nums[k] > 0:
                break
                
            # 去重1: 跳过重复的第一个数
            if k > 0 and nums[k] == nums[k - 1]:
                continue
                
            # 优化2: 当前k的最小可能和已经大于0
            if nums[k] + nums[k + 1] + nums[k + 2] > 0:
                break
                
            # 优化3: 当前k的最大可能和仍然小于0
            if nums[k] + nums[-2] + nums[-1] < 0:
                continue
            
            # 双指针法查找剩下的两个数
            i, j = k + 1, n - 1
            
            while i < j:
                total = nums[k] + nums[i] + nums[j]
                
                if total < 0:
                    i += 1  # 和太小，左指针右移增加和
                elif total > 0:
                    j -= 1  # 和太大，右指针左移减小和
                else:
                    # 找到有效三元组
                    result.append([nums[k], nums[i], nums[j]])
                    
                    # 去重2: 跳过重复的第二个数
                    while i < j and nums[i] == nums[i + 1]:
                        i += 1
                    # 去重3: 跳过重复的第三个数
                    while i < j and nums[j] == nums[j - 1]:
                        j -= 1
                    
                    # 继续寻找其他可能的组合
                    i += 1
                    j -= 1
        
        return result
