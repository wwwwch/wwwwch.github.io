⚙️ 模板步骤

场景动作结论
长度奇数直接返回 False必无效
遇左括号入栈对应右括号等待匹配
遇右括号栈顶弹出比较空栈/不等即 False
遍历结束检查栈栈空才 True
易错点：不是把左括号入栈，而是把“期望的右括号”入栈；这样比较更直接。
代码 V1｜栈 + 映射表（完整保留） 
class Solution:
    def isValid(self, s: str) -> bool:
        if len(s) % 2: # s 长度必须是偶数
            return False

        mp = {'(':')', '[':']', '{':'}'} # 括号映射表
        st = []  # 列表模拟栈
        for c in s:
            if c in mp: # c是左括号
                st.append(mp[c]) # 入栈对应的右括号
            elif (not st) or (st.pop() != c):   
                return False    # 没有左括号，或者左括号类型不对

        return (not st)    # 所有字符处理完后，栈应为空
代码 V2｜栈 + 判断（完整保留） 
class Solution:
    def isValid(self, s: str) -> bool:
        if len(s) % 2:  # s 长度必须是偶数
            return False
        
        st = [] # 列表模拟栈
        for c in s:
            if c == '(':
                st.append(')')
            elif c == '[':
                st.append(']')
            elif c == '{':
                st.append('}')
            elif (not st) or (st.pop() != c):   
                return False    # 没有左括号，或者左括号类型不对
            
        return (not st)   # 所有字符处理完后，栈应为空
