49. 字母异位词分组｜哈希表分组

目标：把互为字母异位词的字符串放到同一组。
HashMap 排序作 key 保留原字符串

核心判断

排序后相等 ⇒ 同一组异位词
aab原字符串
→
aab排序 key
←
aba / baa同 key
记忆：key 是“统一特征”，value 存“原字符串列表”。

示例分组图

注意：返回的组顺序、组内顺序通常不要求固定。

流程速记

1

4

5

关键变量

key
排序后的字符串，如 aet
value
原字符串列表，如 [eat, tea, ate]
本质
同类字符串映射到同一个 key
结果
哈希表所有 value

易错点

不要把 排序后的字符串 加入结果；结果必须保留 原字符串。
需要 defaultdict(list) 或手动初始化空列表。
复杂度：常见写法约 O(m·nlogn)；m 为字符串数，n 为单词最大长度。

完整代码

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        d = defaultdict(list)  # 如果 key 不在字典中，defaultdict 会自动创建一个空列表作为 value
        
	for s in strs:
            sorted_s = ''.join(sorted(s))  # 把 s 排序，作为字典的 key
            d[sorted_s].append(s)  # 排序后相同的字符串，分到同一组

        return list(d.values())  # 哈希表的所有 value 就是分组结果
