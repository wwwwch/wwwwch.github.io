📚 73.矩阵置零 —— 空间优化对比

🎯 问题本质

给定 m×n 矩阵，若某个元素为 0，则将其所在行和列的所有元素都设为 0。原地修改。

📊 方法对比

方法 空间复杂度 实现思路 优缺点
V1：辅助数组 O(m+n) 用两个数组分别标记需要清零的行和列 直观易懂，但需额外空间
V2：常量空间 O(1) 利用第一行和第一列作为标记数组 空间最优，但需特殊处理第一行/列

🔍 V1：辅助数组法 O(m+n)

💡 核心思想

  • 用 row_zero[i] 标记第 i 行是否需要清零
  • 用 col_zero[j] 标记第 j 列是否需要清零

💻 关键代码

def setZeroes(self, matrix: List[List[int]]) -> None:
    m, n = len(matrix), len(matrix[0])
    row_zero = [False] * m
    col_zero = [False] * n

    # 第一遍：标记
    for i in range(m):
        for j in range(n):
            if matrix[i][j] == 0:
                row_zero[i] = True
                col_zero[j] = True

    # 第二遍：置零
    for i in range(m):
        if row_zero[i]:
            matrix[i] = [0] * n      # 整行清零
        else:
            for j in range(n):
                if col_zero[j]:
                    matrix[i][j] = 0

🔍 V2：常量空间法 O(1)

💡 核心思想

  • 用第一行记录哪些列需要清零
  • 用第一列记录哪些行需要清零
  • 额外用两个变量记录第一行/列本身是否需要清零

💻 关键代码

def setZeroes(self, matrix: List[List[int]]) -> None:
    m, n = len(matrix), len(matrix[0])
    first_row_zero = any(matrix[0][j] == 0 for j in range(n))
    first_col_zero = any(matrix[i][0] == 0 for i in range(m))

    # 第一遍：用第一行和第一列做标记（跳过第一行第一列）
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][j] == 0:
                matrix[i][0] = 0
                matrix[0][j] = 0

    # 第二遍：根据标记置零（跳过第一行第一列）
    for i in range(1, m):
        if matrix[i][0] == 0:
            matrix[i] = [0] * n
        else:
            for j in range(1, n):
                if matrix[0][j] == 0:
                    matrix[i][j] = 0

    # 最后处理第一行和第一列
    if first_row_zero:
        for j in range(n):
            matrix[0][j] = 0
    if first_col_zero:
        for i in range(m):
            matrix[i][0] = 0

⚠️ 关键细节

细节 V1 处理 V2 处理
标记方式 独立数组 复用矩阵的第一行和第一列
行清零 直接赋整行0 行标记在 matrix[i][0]
列清零 遍历列标记 列标记在 matrix[0][j]
第一行/列自身 正常标记 需额外变量记录，最后处理
遍历范围 全部 跳过第一行第一列，避免干扰

💡 一句话总结

V1用辅助数组标记行列，V2复用第一行/列作为标记区，额外变量记录首行列状态，空间从 O(m+n) 降至 O(1)。
