三角形最小路径和

自底向上 DP

🎯 问题本质

给定三角形二维列表，从顶部到底部找一条路径，使路径上数字之和最小。

移动规则：每步只能移动到下一行相邻节点，即索引 相同 或 +1。

下一步：triangle[i][j] → triangle[i+1][j] 或 triangle[i+1][j+1]

📌 路径示意

2
3
4
6
5
7
4
1
8
3
最小路径：2 → 4 → 5 → 1 = 12

🔍 核心思想

  • 从底部向上推，底部节点天然是已知答案。
  • 每个位置的最小路径和 = 自身值 + 下方两个相邻子路径和的较小值。
  • 更新到顶部后，triangle[0][0] 即为全局最优解。

🧠 状态定义

dp[i][j] = 从第 i 行第 j 列出发到底部的最小路径和
初始：dp[last][j] = triangle[last][j]
转移：dp[i][j] = triangle[i][j] + min(dp[i+1][j], dp[i+1][j+1])
💻 关键代码：原地修改 空间 O(1) 
def minimumTotal(self, triangle: List[List[int]]) -> int:
    # 从倒数第二行开始向上处理
    for row in range(len(triangle) - 2, -1, -1):
        for col in range(len(triangle[row])):
            # 选择下一行相邻两个中较小的路径和
            triangle[row][col] += min(triangle[row+1][col], triangle[row+1][col+1])
    return triangle[0][0]

📊 复杂度分析

维度性能
时间复杂度O(n²)：n 为行数，每个元素遍历一次
空间复杂度O(1)：原地修改，无额外 DP 数组

顶部答案

⚠️ 关键细节

遍历顺序从倒数第二行开始，row 递减；每行从左到右。
边界处理每行元素数为 row+1；子节点索引为 col 和 col+1。
原地修改直接复用输入数组，节省空间。
非原地版本用 dp = triangle[-1][:] 拷贝最后一行，再向上滚动更新。

方法空间难点推荐

💡 一句话总结：从倒数第二行开始向上更新，每个位置加上下方相邻较小值；最终顶部即为最小路径和，O(n²) 时间，O(1) 空间。
