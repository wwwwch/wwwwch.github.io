📚21.合并两个有序链表 —— 多解法总结

🎯 问题本质

将两个升序链表合并为一个新的升序链表，返回合并后的链表。

🔍 核心思想

双指针遍历两个链表，每次取较小节点接入结果链表，类似合并两个有序数组。

📊 方法对比总览

方法 核心思路 时间复杂度 空间复杂度 是否破坏原链表 代码复杂度
1. 迭代法（新建节点） 创建新节点，值拷贝 O(n+m) O(n+m) ❌ 不破坏 中等
2. 迭代法（复用节点） 修改节点指针，直接拼接 O(n+m) O(1) ✅ 修改原链表 简单
3. 递归法 递归选择较小节点，自底向上构建 O(n+m) O(n+m)（递归栈） ✅ 修改原链表 简洁
注：三种方法均返回新链表的头节点，时间复杂度相同，空间上复用节点法最优，递归法最简洁但受栈深度限制。

🔧 方法详解

1. 迭代法（新建节点）

思路：遍历两个链表，每次比较节点值，创建新节点并赋值，最后处理剩余节点。

适用：要求不修改原链表，或需要保留原始数据。

def mergeTwoLists(list1, list2):
    dummy = ListNode()      # 哨兵节点，简化头节点处理
    cur = dummy             # cur始终指向新链表尾部
    p, q = list1, list2

    while p and q:
        if p.val <= q.val:
            cur.next = ListNode(p.val)  # 创建新节点
            p = p.next
        else:
            cur.next = ListNode(q.val)
            q = q.next
        cur = cur.next

    # 处理剩余节点
    while p:
        cur.next = ListNode(p.val)
        cur = cur.next
        p = p.next
    while q:
        cur.next = ListNode(q.val)
        cur = cur.next
        q = q.next

    return dummy.next

2. 迭代法（复用节点）

思路：直接修改链表节点的 next 指针，将较小节点接入结果链表，最后链接剩余部分。

适用：追求空间效率，允许修改原链表。

def mergeTwoLists(list1, list2):
    dummy = ListNode()      # 哨兵节点
    cur = dummy

    while list1 and list2:
        if list1.val <= list2.val:
            cur.next = list1       # 将list1节点接入结果
            list1 = list1.next
        else:
            cur.next = list2
            list2 = list2.next
        cur = cur.next

    # 直接链接剩余链表
    cur.next = list1 if list1 else list2

    return dummy.next

3. 递归法

思路：比较头节点值，较小者作为新头，其 next 指向剩余部分合并的结果。

注意：递归深度可能超过栈限制（链表过长时）。

def mergeTwoLists(list1, list2):
    # 递归终止条件
    if not list1:
        return list2
    if not list2:
        return list1

    if list1.val <= list2.val:
        list1.next = mergeTwoLists(list1.next, list2)  # 递归合并剩余
        return list1
    else:
        list2.next = mergeTwoLists(list1, list2.next)
        return list2

⚠️ 关键细节提醒

细节 说明 示例/代码
哨兵节点 避免头节点为空时的特殊判断，统一处理 dummy = ListNode()
剩余链表处理 循环结束后，直接将剩余链表接上（复用节点法）或逐个复制（新建节点法） cur.next = list1 or list2
递归终止条件 任一链表为空时，直接返回另一个链表 if not l1: return l2
自赋值问题 新建节点法注意 cur.next 指向新节点后要移动 cur cur = cur.next

💡 一句话总结

“双指针取较小节点接入结果链表，递归或迭代实现，复用节点法空间最优，新建节点法保留原链表。”
