📚 53.最大子数组和 —— 解法总结

🎯 问题本质
给定整数数组 nums，找出一个具有最大和的连续子数组（至少包含一个元素），返回其最大和。
📊 方法对比
方法 核心思路 时间复杂度 空间复杂度 特点
动态规划（标准） dp[i] = max(nums[i], dp[i-1] + nums[i]) O(n) O(n) 直观易懂
动态规划（滚动数组） 用变量 cur 代替 dp 数组 O(n) O(1) 空间优化
前缀和 + 贪心 维护前缀和及历史最小前缀和，ans = max(ans, cur_sum - min_pre) O(n) O(1) 转化思维
🔍 方法详解

1动态规划（标准 + 滚动数组优化）

状态定义：dp[i] 表示以 nums[i] 结尾的最大子数组和。

转移方程：dp[i] = max(nums[i], dp[i-1] + nums[i])

决策逻辑：要么重新开始（只取当前元素），要么接在之前的子数组后面。

滚动优化：用 cur 记录当前结尾最大和，ans 记录全局最大值。

def maxSubArray(self, nums: List[int]) -> int:
    cur = ans = nums[0]
    for i in range(1, len(nums)):
        cur = max(nums[i], cur + nums[i])
        ans = max(ans, cur)
    return ans

2前缀和 + 贪心

核心转化：子数组和 = 前缀和之差。

固定当前前缀和 cur_sum，要得到以当前位置结尾的最大子数组和，需要减去之前出现过的最小前缀和 min_pre。

  • 初始化：cur_sum = nums[0], min_pre = min(0, nums[0]), ans = nums[0]
  • 遍历后续元素：cur_sum += nums[i]
  • 更新答案：ans = max(ans, cur_sum - min_pre)
  • 更新最小前缀：min_pre = min(min_pre, cur_sum)
def maxSubArray(self, nums: List[int]) -> int:
    cur_sum = ans = nums[0]
    min_pre = min(0, nums[0])
    for i in range(1, len(nums)):
        cur_sum += nums[i]
        ans = max(ans, cur_sum - min_pre)
        min_pre = min(min_pre, cur_sum)
    return ans
💡 关键细节
方法 关键点
动态规划 核心在“要么延续，要么重新开始”的决策。
前缀和 + 贪心 维护历史最小前缀和，并处理空子数组：子数组不能为空，所以初始 min_pre 不能直接为 0，需结合第一个元素。
💎 一句话总结
动态规划：以 i 结尾的最大子数组和 = max(自身, 自身 + 前一个结尾最大和)；
前缀和法：当前子数组最大和 = 当前前缀和 - 历史最小前缀和。
两种方法均 O(n) 时间，滚动 DP 可 O(1) 空间。
