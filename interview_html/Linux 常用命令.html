1. 文件与目录操作

用途命令说明
创建目录mkdir dir创建单层目录。
创建嵌套目录mkdir -p dir1/dir2父目录不存在时一并创建。
切换目录cd newdir
cd ..
cd -
cd		分别表示进入目录、返回父目录、回到上一个目录、回到当前用户主目录。
显示当前目录pwd打印当前工作路径。
复制文件cp source_file dest_file复制为新文件。
复制到目录cp file1 file2 dir把多个文件复制进指定目录。
递归复制目录cp -r source_dir dest_dir
rsync -a source_dir/ dest_dir/		rsync -a 适合保留属性并同步目录内容。
创建软链接ln -s linked_file link创建符号链接。
重命名/移动mv source_file dest_file文件、链接、目录均可处理。
删除文件rm file1 file2删除普通文件或链接。
删除目录rmdir dir
rm -rf dir		rmdir 仅删除空目录；rm -rf 强制递归删除，需谨慎。
注意：rm -rf 不进入回收站，尤其避免在根目录、系统目录或变量为空时执行。

3. 查看文件内容

用途命令说明
显示文件内容cat file1 file2适合短文件。
分页查看more file
less file		less 功能更强，可上下翻页、搜索。
查看开头head -10 file显示前 10 行。
查看结尾tail -10 file显示后 10 行。

4. 文件名模式匹配

模式示例含义
*cat *匹配当前目录普通文件名中的任意字符序列。
.*cat .*匹配隐藏文件；注意可能包含 . 和 ..。
*.logcat *.log匹配以 .log 结尾的文件。
*.?ls *.?匹配扩展名为单个字符的文件。

5. 文本处理

用途命令
查找包含子字符串的行grep substring file
忽略大小写搜索grep -i substring file
显示不包含子字符串的行grep -v substring file
递归搜索目录grep -r substring dir
排序文件内容sort file
排序并去重sort -u file
# 示例：查找日志中的错误并分页查看
grep -i "error" app.log | less

6. 权限管理

用途命令说明
给当前用户添加写权限chmod u+w fileu 表示 user。
给所属组添加读权限chmod g+r fileg 表示 group。
给其他用户添加执行权限chmod o+x fileo 表示 others。
给所有用户添加读写权限chmod a+rw filea 表示 all。
让文件/目录可被访问chmod a+rX *X 只给目录或已有执行位的文件加执行权限。
递归修改目录权限chmod -R a+rX dir-R 表示递归。

查找

按名称查找find . -name "log"

基于索引快速查找locate "pub"
find 实时遍历目录；locate 依赖索引，速度快，但可能查不到刚创建的文件。

8. 重定向与管道

用途命令说明
覆盖写入文件ls *.png > image_files目标文件原内容会被覆盖。
追加写入文件ls *.jpg >> image_files把输出追加到文件末尾。
管道传递输出cat *.log | grep error把前一个命令输出作为后一个命令输入。

9. 进程与作业控制

用途命令
显示所有进程ps -ef
实时查看进程资源占用top
终止指定进程kill PID
强制杀死进程kill -9 PID
杀死当前用户全部进程kill -9 -1
点击窗口杀死图形程序xkill
注意：kill -9 是强制终止，进程无法清理临时文件或保存状态；优先尝试普通 kill。

11. 压缩、解压与归档

单文件压缩/解压

格式压缩解压
gzipgzip filegunzip file.gz
bzip2bzip2 filebunzip2 file.bz2

tar 归档

用途命令
创建 gzip 归档tar zcvf archive.tar.gz dir
创建 bzip2 归档tar jcvf archive.tar.bz2 dir

解压 zipunzip archive.zip

tar 常用选项：c 创建，t 列出，x 提取，z gzip，j bzip2，J xz。
